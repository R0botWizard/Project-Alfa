Controlls

<pre>
W,A,S,D - move 
SPACE - jump 
SHIFT - sprint 
RM - aim 
LM - shoot 
1,2,3 - change weapon 
R - reload 
</pre>

<pre>
Win - on 10 enemies killed
Lose - on player death
</pre>
